Você receberá uma lista de tuplas, onde cada tupla contém uma chave e um valor. Sua tarefa é converter essa lista em um dicionário e então utilizar métodos do dicionário para retornar as chaves.

Entrada Uma lista de tuplas, cada tupla consistindo em uma chave (string) e um valor (pode ser qualquer tipo).

Saída: Uma lista com os valores do dicionário.
